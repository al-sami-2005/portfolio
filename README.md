# Al Sami | Developer Portfolio

A high-end, interactive 3D portfolio featuring a custom avatar, smooth GSAP animations, and a modern ML/AI focused aesthetic. Built with React, Vite, and Three.js.

[View Live Portfolio](https://www.alsami.site/)

## ✨ Key Features
- **3D Interactive Avatar**: Custom-designed 3D model integrated with React Three Fiber.
- **Scrollytelling Experience**: Scroll-linked animations using GSAP and Framer Motion.
- **Professional Showcase**: Sections for Projects, Experience, and AI/ML Expertise.
- **Fully Responsive**: Optimized for seamless viewing across all devices.
- **SEO Optimized**: Built-in SEO components and meta tags for better discoverability.

## 🛠️ Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **3D & Animation**: Three.js, React Three Fiber, GSAP, Framer Motion
- **Icons & UI**: React Icons, React Fast Marquee
- **Deployment**: Optimized for Vercel/GitHub Pages

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/al-sami-portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd al-sami-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
Start the development server:
```bash
npm run dev
```

### Building for Production
Create an optimized production build:
```bash
npm run build
```
